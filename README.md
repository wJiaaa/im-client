### 项目介绍

这是一个网页版在线聊天项目，前端使用 Element-plus + Vue3，后端使用nodejs框架nestjs。

### 项目安装

###### 下载安装

```bash
## 克隆项目
git clone https://github.com/wJiaaa/im-client.git
## 安装依赖
pnpm install
# 启动本地开发环境
pnpm dev
```
